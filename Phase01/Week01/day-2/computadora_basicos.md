# Ejercicio de explicacion....

## CPU

La CPU es el cerebro de la computadora, es decir, es aquella que realiza instrucciones y calculos.

## RAM

La RAM es aquella que guarda temporalmente los datos de un proceso.

## Disco

El Disco es aquel que guarda los datos permanententemente.

## Sistema operativo

El Sistema operativo es aquel que controla todo, es que que hace que la computadore funcione.
