# Día 2 — Cómo funciona una computadora

Objetivo del día:  
Entender qué ocurre dentro de una computadora cuando ejecuta un programa.

Hoy aprenderás:  
- CPU
- RAM
- Almacenamiento
- Sistema operativo

Tiempo total: 1 hora

---

## 1. Las 4 partes principales de una computadora

Todas las computadoras tienen 4 partes clave.

- CPU
- RAM
- STORAGE
- SISTEMA OPERATIVO

Vamos una por una.

---

## 2. CPU — El cerebro de la computadora

CPU significa:  
Central Processing Unit (Unidad Central de Procesamiento)

La CPU es el cerebro de la computadora.

La CPU hace tres cosas:

- leer instrucciones
- ejecutar instrucciones
- hacer cálculos

Ejemplo:

Cuando abres un programa como Visual Studio Code:  
La CPU empieza a ejecutar millones de instrucciones.

### Velocidad de la CPU

Se mide en GHz.

Ejemplo:

3.2 GHz

Eso significa aproximadamente:

3.2 mil millones de ciclos por segundo

---

## 3. RAM — Memoria temporal

RAM significa:

Random Access Memory (Memoria de Acceso Aleatorio)

La RAM es memoria temporal.

Cuando abres un programa:  
el programa se carga en la RAM

Ejemplo:

Si abres:

- navegador
- editor de código
- música

Todos esos programas están en la RAM.

### Qué pasa si la RAM se llena

La computadora se vuelve lenta.

Por eso los desarrolladores cierran programas innecesarios.

### Algo muy importante

Cuando apagas la computadora:  
la RAM se borra

Todo lo que estaba en RAM desaparece.

---

## 4. Storage — Disco duro

El almacenamiento es donde se guardan los datos permanentemente.

Ejemplos:
- fotos
- documentos
- programas
- sistema operativo

Tipos comunes:  
HDD  
SSD  

SSD es mucho más rápido.

---

## 5. Sistema Operativo

El sistema operativo controla todo.

Ejemplos:
- Windows
- macOS
- Linux

Tú usas:

Ubuntu

El sistema operativo se encarga de:

controlar programas  
administrar memoria  
comunicarse con hardware  
manejar archivos  

---

## 6. Qué ocurre cuando ejecutas un programa

Supongamos que abres un programa.

Paso 1

el programa está guardado en el disco

Paso 2

el sistema operativo lo carga en RAM

Paso 3

la CPU empieza a ejecutar instrucciones

Proceso completo:

DISCO → RAM → CPU

3. El almacenamiento es donde se guardan los datos permanentemente.
