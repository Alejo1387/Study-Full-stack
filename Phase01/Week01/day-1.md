# Día 1 — ¿Qué es la programación?

## 1. La idea central de la programación

Programar significa:  
Dar instrucciones a una computadora para que haga algo.

Las computadoras no piensan.  
Solo ejecutan instrucciones.

Ejemplo sencillo:

Imagina que le dices a un robot:

1. Camina 3 pasos
2. Gira a la derecha
3. Toma la caja
4. Regresa

Eso es programar.

Un programa es simplemente:  
Una lista de instrucciones que la computadora ejecuta paso a paso.

---

## 2. Cómo funciona cualquier programa

Todos los programas siguen este modelo:  
INPUT → PROCESS → OUTPUT

### Input (entrada)

Son los datos que recibe el programa.

Ejemplos:
- escribir texto
- subir una foto
- ingresar una contraseña
- hacer clic en un botón

### Process (proceso)

El programa trabaja con los datos.

Ejemplos:
- calcular
- guardar información
- verificar contraseña
- ordenar datos

### Output (salida)

Es el resultado que el programa muestra.

Ejemplos:
- mostrar un mensaje
- mostrar una imagen
- enviar un correo
- mostrar resultados

### Ejemplo 1 — Calculadora

Process  
buscar videos en la base de datos

Output

---

## 4. Qué es un programa

Un programa es:  
un conjunto de instrucciones escritas en un lenguaje que la computadora entiende.

Ejemplos de lenguajes:
- Python
- JavaScript
- Java
- C++
- Go

Nosotros más adelante aprenderemos:
- Python
- JavaScript
- React
- FastAPI

---

## 5. Qué hace realmente un programador

Un programador no solo escribe código.

Hace principalmente 3 cosas:

### 1. Resolver problemas

Ejemplo:
- sistema de deudas
- sistema de tareas
- sistema de notas

### 2. Diseñar soluciones

Pensar:  
cómo funciona el sistema

### 3. Escribir las instrucciones

Eso es el código.
