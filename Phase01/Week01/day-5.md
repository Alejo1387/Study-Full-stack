# Día 5 — Qué es el software

Software significa:  
programas

Tipos de software:

---

## Software de sistema

Controla el computador.

Ejemplos:
- Linux
- Windows
- macOS

---

## Software de aplicación

Programas que usamos.

Ejemplos:
- navegador
- Excel
- WhatsApp
- Spotify

---

## Software de desarrollo

Herramientas para programar.

Ejemplos:
- editores
- compiladores
- terminal
