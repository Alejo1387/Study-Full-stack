# Día 3 — Cómo ejecuta instrucciones una computadora

Las computadoras solo entienden:  
0 y 1

Esto se llama:
###### binario

Ejemplo:  
01001000

Eso representa información.

Todo en la computadora es binario:
- texto
- imágenes
- videos
- programas

---

## Qué hace un programa

Un programa es:  
Una lista de instrucciones

Ejemplo humano:

Receta de cocina.
1. Hervir agua
2. Agregar pasta
3. Cocinar 10 minutos

Eso es un algoritmo.

---

## Qué es un algoritmo

Un algoritmo es:  
una serie de pasos para resolver un problema

Ejemplo:

Algoritmo para hacer café.

1. Calentar agua
2. Poner café en taza
3. Verter agua
4. Revolver
