# Día 2 — Cómo funciona una computadora

Las computadoras tienen 4 componentes principales.

---

### CPU

###### CPU significa:

Central Processing Unit

Es el cerebro de la computadora.

###### La CPU:

ejecuta instrucciones

hace cálculos

controla procesos

Cuando ejecutas un programa:

CPU ejecuta millones de instrucciones por segundo

---

### RAM

###### RAM significa:

Random Access Memory

Es la memoria temporal.

Ejemplo:

Cuando abres un navegador:

El programa se carga en la RAM

Si apagas el computador:

La RAM se borra

---

### Disco duro (Storage)

Aquí se guarda todo de forma permanente.

Ejemplo:

- fotos
- programas
- documentos
- sistema operativo
- Tipos comunes:
- HDD
- SSD

---

### Sistema Operativo

El sistema operativo controla la computadora.

Ejemplos:

- Windows
- macOS
- Linux

Tú usas:

Linux Ubuntu.

El sistema operativo:

- administra memoria
- administra programas
- controla hardware
